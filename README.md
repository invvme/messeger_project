Проект будет реализовывать обмен сообщениями между пользователями в реальном времени с поддержкой файлов и уведомлений через бот


Функции:
- Регистрация пользователя
- Авторизация
- Выход из системы

Чаты:
Создание личных чатов между пользователями
Отправка сообщений в чат
Отображение истории сообщений
Метка времени сообщений
Отображение автора сообщения

Профиль пользователя:
Аватар
Имя пользователя

Интерфейс:
Использование Bootstrap
Список чатов
Окно переписки

Хранение данных:
Хранение всех сообщений в базе данных
Хранение файлов аватарок на сервере
Хранение информации о пользователе

API:
перевод сообщений: LibreTranslate API (может измениться)

Возможные функции:
Поиск по сообщениям
Статус пользователя (онлайн/офлайн)
Уведомления в браузере
Тёмная тема
Удаление сообщений
Редактирование сообщений

Используемые технологии:
Python 
Flask
Flask-Login
Flask-SQLAlchemy
Bootstrap 5
SQLite
HTML CSS
REST API
